# 天玑个人基金组合分析与回测系统 V1.0 最终已知限制清单

## 数据与估值

1. xalpha 及其上游公开数据源可能因网络、接口或页面变化而更新失败。
2. 远程失败时使用最近一次成功保存的真实缓存，缓存数据不代表当日已经更新。
3. 基金正式净值存在披露时间差；QDII 通常比境内基金滞后一个或多个交易日。
4. 详情页补全仅使用已正式公布的单位净值，不使用盘中估算值；累计净值未同步时，累计收益指标使用最近一条累计净值完整记录。
5. 组合净值截止到所有有效成分共同拥有的最新正式净值日，因此可能早于部分境内基金的最新单位净值日期。
6. 历史持仓缺少真实确认份额时只能估算，并以“估算”或“待估值”标识。

## 分析与回测

1. 组合历史净值按当前持仓权重合成，不还原历史每个时点的真实动态持仓变化。
2. 回测未完整模拟申购费、赎回费、滑点、限购、暂停申购、到账延迟及全部交易执行约束。
3. 定投模拟和回测结果基于历史净值，不等同于真实交易收益，也不代表未来表现。
4. 策略建议为固定人工规则模型，不是机器学习预测或未来价格预测。
5. 风险等级和置信度是软件内部分析标签，不替代基金正式风险评级。

## 交易与数据管理

1. 卖出交易采用 FIFO 批次处理，V1.0 不能直接撤销卖出交易，需要通过更正记录处理。
2. 用户应自行核对基金代码、确认日期、金额、净值、份额和手续费。
3. 当前仓库随附模拟演示数据；新建空白数据文件后不自动载入演示持仓。

## 运行环境

1. GUI 主要面向 Windows 10/11，文件和目录打开功能依赖 Windows。
2. V1.0 需要 Python 3.10 及以上版本及 `requirements.txt` 中的依赖。
3. V1.0 尚未提供独立安装包、完整自动化测试体系和持续集成流程。

## 适用范围

本软件仅用于个人基金数据分析、学习记录、组合分析和回测辅助，不提供自动交易，不构成投资建议、收益承诺或交易指令。

# 天玑个人基金组合分析与回测系统 V1.0 版本修改记录

## 版本信息

- 软件全称：天玑个人基金组合分析与回测系统 V1.0
- 软件简称：天玑基金组合分析系统
- 版本号：V1.0
- 阶段：第三阶段，V1.0 阻断问题处理
- 完成日期：2026-06-12

## 申报可靠性修改

1. 建立统一持仓估值链路，总览、持仓明细、CSV 和策略建议使用同一估值结果。
2. 修复最新正式单位净值补入后累计净值缺失导致收益风险指标为 `NaN` 的问题。
3. 修复 QDII 等披露滞后基金在组合净值中被向前填充为零收益的问题，组合截止到所有有效成分共同拥有的最新正式净值日。
4. 主动重新分析时强制尝试远程更新，远程失败后才回退真实缓存。
5. 增加最新正式公布单位净值补全，不使用盘中估算净值，不推算累计净值。
6. 增加组合零有效权重保护。
7. 完整分析增加“成功、部分完成、失败”状态，避免部分输出失败时仍提示全部成功。
8. 交易、买入批次和历史持仓录入改用基金代码关联；禁止重复基金名称；编辑基金时代码只读。
9. 手工持仓入口改为“历史持仓估算录入”，并显示待估值和数据质量提示。
10. 数据文件不存在时创建空白用户数据，不自动载入另一套内置持仓。
11. 当前仓库演示持仓增加 `data_profile=demo` 标识，GUI 页眉明确显示模拟演示数据。

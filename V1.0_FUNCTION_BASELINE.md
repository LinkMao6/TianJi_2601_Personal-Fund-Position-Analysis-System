# 天玑个人基金组合分析与回测系统 V1.0 功能基线清单

## 1. 软件标识

- 软件全称：天玑个人基金组合分析与回测系统 V1.0
- 软件简称：天玑基金组合分析系统
- 版本号：V1.0
- 主要运行环境：Python 3.10 及以上版本，Windows 10/11

## 2. 申报定位

本软件是面向个人用户的基金数据分析、学习记录、组合分析与回测辅助工具。软件不提供自动交易功能，不连接交易账户执行买卖，不承诺投资收益。

## 3. V1.0 冻结原则

自本基线确定之日起，V1.0 暂停非必要功能扩展。进入申请准备阶段后，仅处理以下事项：

1. 导致软件无法启动或核心流程无法完成的问题。
2. 明显的计算错误或跨页面数据不一致。
3. 模拟数据被误认为真实结果的问题。
4. README、用户文档与实际功能明显不一致的问题。
5. 软著材料所需的软件名称、版本、入口和展示一致性问题。

不影响核心流程的增强需求、体验优化及扩展功能进入后续版本评估。

## 4. V1.0 功能基线

### 4.1 软件入口

- Tkinter 桌面 GUI。
- 命令行完整分析入口。

### 4.2 基金池管理

- 添加、编辑、启用、停用和删除基金。
- 维护基金代码、名称、分类、类型、数据源和定投配置。
- 删除基金时处理该基金关联的持仓、交易和批次记录。

### 4.3 数据源与缓存

- 通过 xalpha 获取基金信息、单位净值和累计净值。
- 普通基金和货币基金使用相应的数据获取方式。
- 将成功获取的真实净值写入本地缓存。
- 远程更新失败时使用已有有效缓存。
- 无远程数据且无可用缓存时显示明确错误或数据不足。
- 展示数据来源、净值日期、缓存状态和错误信息。

### 4.4 持仓、交易与批次

- 维护基金级持仓汇总。
- 登记确认买入和卖出交易。
- 管理同一基金的多笔买入批次。
- 支持批次添加、编辑和删除。
- 根据交易和剩余批次重建持仓。
- 对历史缺失份额的数据进行估算并标记数据质量。
- 对旧版持仓数据执行版本迁移并保留备份。

### 4.5 统一估值

- 根据剩余份额和最新净值计算当前市值。
- 计算持仓成本、收益、收益率和组合权重。
- 汇总组合总市值、总成本和累计收益。
- 总览、持仓明细、CSV、报告和策略建议复用统一估值结果。

### 4.6 收益风险与组合分析

- 计算阶段收益。
- 计算年化收益、年化波动率、最大回撤、夏普比率和卡玛比率等指标。
- 计算 RSI、移动平均线和近期回撤等量化指标。
- 生成资产配置分析。
- 生成组合净值及组合级风险指标。

### 4.7 定投与策略回测

- 固定定投模拟。
- 基于 MA60 的动态定投回测。
- 基于 MA20 和 MA60 的均线策略回测。
- 支持设置基金、日期范围、频率和基础金额。
- 输出投入、期末资产、收益、风险指标、交易次数和策略资金曲线。

### 4.8 策略建议

- 根据趋势、位置、风险、持仓和基金配置生成规则化建议。
- 支持普通基金、定投基金和特殊基金规则。
- 生成定投倍率和建议金额。
- 生成止盈、减仓和止损风险提示。
- 展示建议理由、数据质量和置信度。
- 保存建议历史并导出 Markdown 报告。

### 4.9 GUI 展示

- 总览与图表。
- 持仓明细。
- 交易流水。
- 持仓批次。
- 基金池管理。
- 数据中心。
- 收益与风险。
- 定投模拟。
- 量化指标。
- 策略回测。
- 策略建议。
- 运行日志。

### 4.10 输出与日志

- 生成组合摘要、持仓、阶段收益、风险、指标、组合净值、回测和定投等 CSV。
- 生成资产配置、类别收益和风险比较等 PNG 图表。
- 生成组合分析及策略建议 Markdown 报告。
- 记录数据源、缓存、净值日期、估值方式、分析过程和异常信息。

## 5. 已知限制

最终版本以 [`V1.0_KNOWN_LIMITATIONS.md`](V1.0_KNOWN_LIMITATIONS.md) 为准。

1. xalpha 依赖第三方公开数据源，网络异常或上游页面变化可能导致更新失败。
2. 使用有效缓存时，重新分析不保证净值截止日期发生变化。
3. QDII 基金净值通常比境内基金滞后一个或多个交易日。
4. 历史持仓缺少真实确认份额时只能根据已有数据估算，并以数据质量标签标识。
5. 回测未完整模拟申购费、赎回费、滑点、限购、暂停申购、到账延迟及全部交易约束。
6. 策略建议属于固定规则模型，不是机器学习预测，不预测未来价格。
7. 卖出交易涉及 FIFO 批次，当前不能直接撤销，需要通过更正记录处理。
8. GUI 文件打开等部分功能主要适配 Windows 环境。
9. V1.0 尚未提供完整自动化测试、持续集成和独立安装包。

上述限制原则上通过用户手册、设计说明书和风险提示予以明确；除非其导致软件无法运行、结果明显错误或文档与实际功能严重不一致，否则不作为 V1.0 申报前的强制修复项。

## 6. 统一免责声明

本软件仅用于个人基金数据分析、学习记录、组合分析和回测辅助，不提供自动交易功能。所有收益、风险、回测和策略建议均基于历史数据、公开数据及人工规则，仅供研究参考，不构成任何投资建议、收益承诺或交易指令。历史数据和回测结果不代表未来表现，使用者应自行核对数据、独立作出判断并承担使用风险。
